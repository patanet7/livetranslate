Loopback Page Display Modes

How should live captions + translations appear during a meeting?

A

Subtitle Overlay

Captions pinned to bottom — original language on top, translation below. Minimal, like watching a subtitled video. Great for screen-sharing.

Subtitle Mode
[ Meeting content / shared screen visible here ]
🎤 这是一个关于人工智能的演示文稿
🌐 This is a presentation about artificial intelligence
🎤 我们今天要讨论三个主要话题
🌐 We will discuss three main topics today
B

Meeting Transcript

Scrolling feed with speaker names, timestamps, original text + translation. Like a live meeting transcript. Best for note-taking and review.

Transcript Mode
Speaker 1 (张伟) 14:32:05
这是一个关于人工智能的演示文稿
This is a presentation about artificial intelligence
Speaker 2 (李明) 14:32:12
我们今天要讨论三个主要话题
We will discuss three main topics today
Speaker 1 (张伟) 14:32:18
第一个话题是关于...
The first topic is about...
C

Split View

Left panel: original language captions scrolling. Right panel: translations scrolling. Side-by-side comparison. Good for language learning.

Split Mode
Original (中文)
张伟: 这是一个关于人工智能的演示文稿
李明: 我们今天要讨论三个主要话题
Translation (English)
Zhang Wei: This is a presentation about artificial intelligence
Li Ming: We will discuss three main topics today
D

Flexible — All Modes

Toggle between subtitle, transcript, and split modes. Start with transcript as default. Add overlay mode for screen-sharing. User picks what works for the moment.

Includes all three modes above with a mode switcher in the toolbar. Default: Transcript mode. Can pop out subtitle overlay as a separate window.
