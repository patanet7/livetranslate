# Orchestration Components

## Source Layout

- `modules/orchestration-service/src/routers`: API routing surface.
- `modules/orchestration-service/src/services`: service orchestration logic.
- `modules/orchestration-service/src/pipeline`: pipeline coordination.
- `modules/orchestration-service/src/audio`: audio processing stages and orchestration hooks.
- `modules/orchestration-service/src/database`: persistence layer and integration points.
- `modules/orchestration-service/src/gateway`: downstream service communication.
- `modules/orchestration-service/src/worker`: background task/worker flows.
