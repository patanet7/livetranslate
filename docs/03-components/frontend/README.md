# Frontend Components

## Source Layout

- `modules/frontend-service/src/pages`: route-level screens.
- `modules/frontend-service/src/components`: reusable UI/components by domain.
- `modules/frontend-service/src/hooks`: shared behavior and service bindings.
- `modules/frontend-service/src/store`: Redux store, slices, API integration.
- `modules/frontend-service/src/utils`: utility and processing helpers.
- `modules/frontend-service/src/types`: TypeScript domain types.
