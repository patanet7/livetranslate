# Whisper Components

## Source Layout

- `modules/whisper-service/src/transcription`: transcription engine flows.
- `modules/whisper-service/src/audio`: audio preprocessing and handling.
- `modules/whisper-service/src/language_id`: language identification logic.
- `modules/whisper-service/src/session`: session state handling.
- `modules/whisper-service/src/orchestration`: integration-facing adapters.
- `modules/whisper-service/src/config`: service configuration.
